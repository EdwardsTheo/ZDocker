# List of useful kubernetes and minikube commands 

## Minikube Basic 

- 

### Minikube lifecycle
```
minikube start
```

```
minikube delete
```

```
minikube status
```

```
minikube stop
```

## Check status 

### Get minikube version
```
minikube version
```

### Get service info
```
minikube service <service_name>
```

### Display the Dashboard
```
minikube dashboard
```

### Display minikube ip 
```
minikube ip
```

### Display minikube addons
```
minikube addons list
```

### Add an addon
```
minikube addons enable <name_addons>
```
